31//! `wasm4games` — a branchless, `no_std`, engine-agnostic *game-pattern foundry*.
32//!
33//! Game behaviors (input admission, ticks, collision, combat, status, physics, evidence,
34//! promotion) are expressed as **bounded byte-class state** transformed by **branchless
35//! kernels** that reuse the verified primitives in [`bcinr_logic`]. Every meaningful
36//! transition can emit **object-centric evidence** ([`evidence::ocel`]), **runtime span
37//! codes** ([`evidence::otel`]), a **receipt chain** ([`evidence::receipt`]), and a
38//! **deterministic replay frame** ([`evidence::replay`]).
39//!
40//! # Quick Start
41//!
42//! Call a pattern kernel directly for zero-allocation, branch-free game logic:
43//!
44//! ```
45//! use wasm4games::patterns::damage_applied;
46//!
47//! // state: bits[0..16] = current HP (200)
48//! // input: bits[0..16] = base damage (25), bit[16] = crit flag (0 = no crit)
49//! let new_state = damage_applied(200, 25);
50//! assert_eq!(new_state, 175);
51//!
52//! // With a crit (bit 16 set), damage is doubled.
53//! let crit_state = damage_applied(200, 25 | (1 << 16));
54//! assert_eq!(crit_state, 150);
55//! ```
56//!
57//! # Pattern Kernels
58//!
59//! All 75 pattern kernels live in [`patterns`]. Each kernel is a pure function
60//! `(state: u64, input: u64) -> u64` using a packed-u64 ABI. The complete catalog
61//! is exposed via [`patterns::PATTERN_REGISTRY`] as [`ir::PatternSpec`] data, which
62//! allows iteration, self-checking, and portability verification without any runtime
63//! reflection.
64//!
65//! # Evidence Channels
66//!
67//! Kernels are intentionally free of side effects; evidence is emitted by the caller
68//! using the four channels in [`evidence`]:
69//!
70//! | Module | Purpose |
71//! |---|---|
72//! | [`evidence::ocel`] | Object-centric event log (OCEL-style) |
73//! | [`evidence::otel`] | Runtime span codes |
74//! | [`evidence::receipt`] | Deterministic rolling receipt chain |
75//! | [`evidence::replay`] | Deterministic replay frames |
76//!
77//! # No-std Usage
78//!
79//! The crate is `#![no_std]` by default. Enable the `alloc` feature for growable
80//! structures like [`evidence::ocel::OcelLog`]. Enable the `std` feature if you need
81//! the full standard library in tests or integration code.
82//!
83//! ## Doctrine
84//!
85//! - Engines project worlds; `wasm4games` operates patterns; `wasm4pm` admits evidence.
86//! - Kernels are **generated by ggen** from the input surface in `ggen/`; the committed
87//!   files under [`patterns`] are the offline source of truth. See the "GGEN-ONLY USER
88//!   SURFACE" covenant in the docs.
89//! - `wasm4pm` is *referenced, not vendored*: [`compat`] holds dependency-free mirror
90//!   types shaped like `wasm4pm-compat`; the real admission bridge lives in the
91//!   workspace-excluded `wasm4games-wasm4pm` crate.
