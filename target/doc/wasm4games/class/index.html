Byte-class authority: bounded u8 state alphabets and the admission status vocabulary. Byte-class authority: bounded u8 state alphabets and the admission status vocabulary.

The whole crate operates on small, bounded u8 domains so that kernels remain branchless and SIMD-friendly. Joining and clamping lower onto [bcinr_logic] primitives ([bcinr_logic::mask::max_u32], [bcinr_logic::fix::clamp_u32]) rather than if/match.

§Status lattice

[status] constants form a total order whose numeric value is the “severity” of the observation. This lets a branchless max compute the worst-of join across an event window. Terminal/abnormal codes (REFUSED, RESIDUAL) sort above the normal lifecycle so they dominate a join.

UNKNOWN(0) < BLOCKED(1) < PARTIAL(2) < PENDING(3) < ADMITTED(4)
          < PROJECTED(5) < RECEIPTED(6) < REFUSED(7) < RESIDUAL(8)
