pub fn choice_weight_selected(state: u64, input: u64) -> u64
Expand description

ChoiceWeightSelected kernel. Packed-u64 ABI:

  • state: bits[0..8] = roll (0..255).
  • input: bits[0..8] = b0, bits[8..16] = b1, bits[16..24] = b2 — three bucket boundaries.

Returns: bits[0..8] = choice index (0 if roll < b0, 1 if b0<=roll<b1, 2 if b1<=roll<b2, 3 if roll>=b2).
