Pattern: InventoryItemChanged

§Branchless Contract

Ensures: result matches the reference oracle for all inputs. Invariant: execution path is independent of input data values (Branchless).

§Timing contract

  • T0 primitive budget: ~1 ns. T1 aggregate: <= 200 ns. Max heap allocations: 0.

Change an inventory: on add, find the first free slot in the occupancy bitset with [bcinr_logic::bitset::select_bit_u64] over the inverted mask and occupy it; on remove, clear a given slot. A full inventory reports the all-ones sentinel slot 0xFF.

Constants§

NO_SLOT
Sentinel returned in the slot field when no free slot exists.
