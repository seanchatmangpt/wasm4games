Pattern: SyncStateAdmitted

§Branchless Contract

Ensures: result matches the reference oracle for all inputs. Invariant: execution path is independent of input data values (Branchless).

§Timing contract

  • T0 primitive budget: ~1 ns. T1 aggregate: <= 200 ns. Max heap allocations: 0.

Advance a synchronization FSM via branchless DFA table lookup. States: UNSYNCED=0, SYNCING=1, SYNCED=2, DRIFTED=3, LOST=4 (5 states). Symbols: START_SYNC=0, ACK=1, DRIFT=2, TIMEOUT=3, RESYNC=4 (5 symbols).
