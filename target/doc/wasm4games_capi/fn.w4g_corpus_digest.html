#[unsafe(no_mangle)]
pub extern "C" fn w4g_corpus_digest() -> u64
Expand description

Recomputes the full corpus digest from this C-linked build.

Folds every pattern kernel over its test corpus and produces a single u64 rolling hash. Compare against w4g_golden_corpus_digest to verify cross-language equivalence.

§Safety

No pointer arguments. Pure computation; safe to call from any C context.

§Returns

A u64 digest reflecting every registered kernel’s output over the shared corpus. This value should equal w4g_golden_corpus_digest on a correct build.
