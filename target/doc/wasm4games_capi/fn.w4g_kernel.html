Expand description

Dispatches a single pattern kernel by id over a packed-u64 (state, input) ABI.

§Safety

No pointer arguments. All three parameters are passed by value; there are no lifetime or aliasing constraints. Safe to call from C as long as pattern_id is within [0, w4g_pattern_count()). Out-of-range ids are handled defensively by [wasm4games::corpus::dispatch].

§Returns

The packed u64 output of the requested kernel. Semantics are kernel-specific; see the wasm4games pattern registry for field layout.
