#[unsafe(no_mangle)]
pub extern "C" fn w4g_pattern_count() -> u32
Expand description

Returns the number of registered pattern kernels.

Use this to validate pattern_id bounds before calling w4g_kernel.

§Safety

No pointer arguments. Safe to call from any C translation unit that links the wasm4games-capi staticlib. This function follows the platform C calling convention (extern "C").

§Returns

The count of all kernels in the wasm4games pattern registry. A return value of 0 indicates an empty registry and should be treated as a configuration error.
